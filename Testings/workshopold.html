What is this workshop about?

This workshop covers various concepts of mobile communication and microcontroller programming, necessary for the development of a mobile-controlled robot. DTMF stands for "Dual tone multi frequency." We will be using a dedicated modem/mobile at the receiver module, i.e., with the robot itself, and send the commands using DTMF tones at the receiver's end. Conventionally, Wireless-controlled robots use radio control, which has limitations like limited working range, limited frequency range, and interference. A mobile-controlled robot overcomes these limitations, allowing remote operation via a mobile phone.
